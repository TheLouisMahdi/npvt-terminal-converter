## Overview

**NPVT Terminal Converter** is a browser-based converter built as one portable HTML file. It is designed for local use, with no installation, no backend server, and no upload step.

The project is made for users who want a clean offline tool that can be downloaded once and opened directly in a browser.

---

## Features

| Feature | Description |
|---|---|
| Single-file app | Everything is inside `index.html`. |
| Local workflow | Files are processed in the browser. |
| Offline use | Works without a server after download. |
| Drag and drop | Import supported files directly. |
| Copy / download output | Export converted results easily. |
| Desktop and mobile UI | Responsive green terminal-style interface. |

---

## How to Use

1. Click the **Download index.html** button above.
2. Open the downloaded file in your browser.
3. Import or paste your content.
4. Convert locally.
5. Copy or download the output.

No setup is required.
