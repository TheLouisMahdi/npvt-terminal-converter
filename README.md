# NPVT Terminal Converter

A single-file offline HTML converter.

## Download

**Main file:** [`index.html`](./index.html)

Direct download:

```text
https://github.com/TheLouisMahdi/npvt-terminal-converter/raw/main/index.html
```

## Usage

1. Download `index.html`.
2. Open it in your browser.
3. Use the converter locally.

No installation is required.
