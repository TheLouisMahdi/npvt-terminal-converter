# Security Policy

## Local-Only Design

This project is designed to run fully inside the browser. It does not require a backend server and does not upload files or generated output anywhere.

## Data Safety

VPN and proxy configuration files may include private connection details. Avoid sharing real configuration files publicly in issues, screenshots, commits, or pull requests.

## Reporting Issues

If you find a security-related issue, open a private report through GitHub Security Advisories if available, or contact the maintainer directly through GitHub.

## Recommended Practice

Before publishing screenshots or examples, replace real connection details with dummy values.
