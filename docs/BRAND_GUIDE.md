## Visual Identity

Use a clean technical identity across the repository and preview graphics.

## Color Palette

| Token | Hex | Usage |
|---|---:|---|
| Black | `#020704` | Main background |
| Deep green | `#06140b` | Panels and shadows |
| Green | `#39ff88` | Main accent |
| Signal green | `#00d46a` | Secondary accent |
| Mint | `#b8ffd2` | Highlights and text |
| Muted green | `#82ad92` | Secondary text |
| Amber | `#ffcb6b` | Warnings |
| Pink | `#ff6688` | Alerts |

## Typography

- UI text: clean sans-serif
- Technical labels: monospace
- Buttons and chips: uppercase monospace

## Repository Rules

- Keep the title as the project name only.
- Put details inside the README sections, not in the main title.
- Use badges only for real project traits.
- Avoid fake official logos.
- Avoid aggressive hacker visuals.
