# Contributing

Contributions are welcome.

## Development Style

This project is intentionally kept as a single-file offline HTML app. Please avoid adding external CDN dependencies unless there is a strong reason.

## Pull Request Checklist

- Keep the interface in English.
- Keep the app usable offline.
- Preserve the terminal-inspired green theme unless the change is specifically about themes.
- Test file selection, drag-and-drop, copy, download, clear, and keyboard shortcuts.

## Good First Contributions

- Improve parser compatibility.
- Add clearer error messages.
- Improve mobile layout details.
- Add more export formats.
- Add safe sample configs with dummy values only.
