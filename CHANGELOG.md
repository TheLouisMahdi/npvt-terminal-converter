# Changelog

All notable changes to this project will be documented in this file.

## 1.0.0 - 2026-07-04

### Added

- Initial single-file HTML release.
- Offline NPVT / JSON / subscription to V2Ray link conversion.
- Reverse conversion from V2Ray links to an encrypted NPVT1-style bundle.
- Green terminal-inspired interface.
- Animated right-side bubble menu.
- Drag-and-drop file handling.
- Copy and download actions.
- Clear actions for outputs and tables.
- Enter-to-convert keyboard shortcut.
